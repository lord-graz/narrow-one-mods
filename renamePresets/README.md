# Rename Presets  
A JS script that adds the missing functionality of naming your presets. This allows you to find your presets easier (especially if you have a lot of them), or just to keep them more organized.\
It does this by modifying the IndexedDB database of presets, so you can also change the names by using other scripts (such as IndexedDBEdit).
