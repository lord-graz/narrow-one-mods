# Narrow.One Mods
#### by Lord Graz

This is just a place for me to share my Narrow.One coding creations. Usually they are custom UI CSS code or mild JS mods (only DOM modifications).

# Current public mods:
## <a href="narrowGlass">NarrowGlass</a>  
Pure CSS mod that makes the UI feel like glass, loosely inspired by Apple's LiquidGlass theme. Availible with the top left corner of dialogs sharp (narrowGlassSharp) or round (narrowGlassRound). The Image is a collection of screenshots from narrowGlassSharp.
> [!NOTE]
> Certain parts of this mod might not work in non-WebKit (e.g. Firefox) browsers.  

<img src="/_img/narrowGlass.png" width="400" style="border-radius: 20vw"/>  

## <a href="renamePresets">Rename Presets</a>  
A JS script that adds the missing functionality of naming your presets. This allows you to find your presets easier (especially if you have a lot of them), or just to keep them more organized.

<img src="/_img/renamePresets01.png" width="400" /> <img src="/_img/renamePresets02.png" width="300" />
  
## <a href="showItemNameInTooltip">Show Item Name in Tooltip</a>  
JS script to show the name of shop items in their stat tooltip. This can be useful if you want to switch armor/weapons midgame, but the 3D models (needed for you to see what you are equipping) load too slowly, or if you're just interested in easily seeing what the names of the shop items are :) .  
  
<img src="/_img/showItemNameInTooltip.png" width="400" style="border-radius: 20vw"/>  
  
## <a href="rotatedQuadCrosshair">Rotated Quad Crosshair</a>  
A simple custom crosshair that is similar to crosshairs from other fps games. The lines still act in the same way as you would expect from a regular narrow one crosshair.
It is availible with a dot, without a dot, as JavaScript or CSS, as you prefer.  
> [!TIP]
> For best results change crosshair type in the Narrow.One settings to "QUAD".
  
<img src="/_img/rotatedQuadCrosshair.png" width="400" style="border-radius: 20vw"/>  

## <a href="favoriteShopItems">Favorite Shop Items</a>  
  
This mod allows you to pin shop items to the top of the shop. It can make it easier to find what you are looking for, as often you have to scroll down through a long list of items in the shop.
